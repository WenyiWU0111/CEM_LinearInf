Usage
=====
.. _dependencies:

Dependensies
------------
* ``numpy >= 1.25.1``
* ``pandas >= 1.4.1``
* ``scikit-learn >= 1.3.0``
* ``scipy >= 1.11.1``
* ``statsmodels >= 0.14.0``
* ``matplotlib >= 3.5.0``
* ``seaborn >= 0.12.2``



.. _installation:

Installation
------------

To use ``CEM_LinearInf``, please install it using ``pip``:

.. code-block:: console

   (.venv) $ pip install CEM_LinearInf
